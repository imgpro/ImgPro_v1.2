+ Работает с *.img архивами  GTA3(VER1), Vice City(VER1) и San Andreas(VER2).
+ Возможность заменять/добавлять несколько файлов за раз.
+ Возможность извлекать файлы

+ Исправлено: прграмма неработала без некоторых библиотек DLL

=============================================================================



В нижней строчке надо указать путь к *.img файлу. Для этого надо нажать
кнопку  Browse...  и указать путь к файлу.


Кнопка Add... добавляет файлы в список


Extract...   - извлекает в указанную папку файлы с теми именами,
которые есть в списке

Replace      - добавляет в IMG архив файлы из списка, если файл
уже есть в архиве, он заменяется



Чтобы добавить файлы в список необязательно использовать кнопку Add...
можно просто перетащить нужные файлы в окно программы
